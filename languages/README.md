# languages

**In this directory are the translation files of your application and the routes.**

---

The translated text must have the same index in all translation files. Example:


## Text translation

File: **en/app.json**

```json
{
  "A0001" : "Hello World!",
  "A0002" : "Welcome",
  "A0003" : ""
}
```
